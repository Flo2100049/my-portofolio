# Project Portofolio

Υλοποιηση του Portofolio Site μου, με front κομματι σε HTML, CSS, JS. Backend κομματι το οποιο θα ειναι Python API με την
Flask Βιβλιοθηκη η οποια θα εποικεινωνει με την βαση μας και θα στελνει τυχον events στο RabbitMQ, και για βαση δεδομενων μια MongoDB. Επισης και ενα Microservice για το notification event κομματι.
